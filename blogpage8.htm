Back

07/02/2025 at 13:24

Hi all, sorry for the recent absence! Mock results have come in (so I was caught up with that) but in the mean time I think i've improved the r-space somewhat in terms of connections.

One of the reasons I use Microsoft FrontPage 2000 is because it means I can load it on older web browsers. One of which being my PSP, which loaded great on it's web browser! 

This remained until recently where I noticed some people were complaining about the fact that on loading the website for the first time, Chrome reported that 'This website is insecure' and 'attackers could be trying to capture sensitive information' (despite the fact that this website has no form of data exchange). So I got an SSL certificate for it, which worked! The issue with it is that I could no longer load the website on my PSP (due to Sony's infinite wisdom and using only SSL 3.0 for the PSP's browser) which is unacceptable in my mind. It would be taking one step forward and two steps back.

Luckily, I found a way (i think) to support both secure and insecure connections. For devices without modern HTTPS support (PSP included), it can still connect to the website using regular HTTP. For any other device, it connects securely! Mission accomplished.

Though even still, If you have any older devices (laptops, phones, game consoles, etc.) I would really appreciate if you try to connect and load this website using them and send me an email or DM with your results! You can reach me via my email (rspace@rspace.co.uk) or via my Discord, which is simply 'rspace'. Feedback on the website is also appreciated :)

Cheers and happy Friday!
